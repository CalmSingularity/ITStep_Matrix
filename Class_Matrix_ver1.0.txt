 	Задача "Класс Matrix" (вер. 1.0).
 
	Создать класс "Матрица" размером N x M (определяется через конструктор). Определить конструкторы копирования и 
перемещения. Интерфейс доступа к элементам -- M (i, j).
	Операции:
          - Сложение матриц
          - Вычитание матриц
          - Умножение матрицы на число
          - Умножение матриц
          - Возведение матрицы в натуральную степень
              При степени равной нулю -- единичная матрица
              При степени равной -1   -- обратная (если существует)
	  - Вычисление определителя матрицы
	  - Вычисления следа матрицы
	Арифметические операции должны быть реализованы в виде перегруженных операторов, если возможно.
	Должен быть определен(ы) оператор(ы) записи объекта в поток вывода -- operator<<
